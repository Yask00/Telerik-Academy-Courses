* Collision detection
* Different shooting types
  * Bullets
  * Lasers (just visual for now)
* Animated background (space-like)
* Enemy movements
* Different enemy types
  * Bonuses
  * Walls
* Health and score HUD
* Save high-scores in local storage (maybe on a server)
